Easy Trigger

Timecode-driven Resolume Clip Trigger – User Guide Тригер кліпів Resolume на основі тайм-коду – Посібник користувача
Easy Trigger receives a timecode signal (LTC, MTC, Art-Net, or OSC), compares the current position against a list of configured clip trigger times, and fires OSC commands to Resolume Arena / Avenue to play the matching clips at the right moment. Each clip has an editable trigger time, optional countdown, duration, and an end-of-clip action. Custom Triggers allow firing arbitrary Resolume columns or layer/clip combinations without a clip list.
Easy Trigger отримує сигнал тайм-коду (LTC, MTC, Art-Net або OSC), порівнює поточну позицію зі списком налаштованих часів тригера кліпів та надсилає OSC-команди до Resolume Arena / Avenue для відтворення відповідних кліпів у потрібний момент. Кожен кліп має редаговані час тригера, необов'язковий зворотний відлік, тривалість та дію після завершення. Власні тригери дозволяють запускати довільні стовпці Resolume або комбінації шар/кліп без списку кліпів.

Генерує аудіосигнал LTC з розкодованого тайм-коду та надсилає його до вибраного аудіовихідного пристрою. Коли активний режим Thru, вхідне аудіо передається безпосередньо без перетворення.

FieldПолеDescriptionОпис
Out LTC toggleПеремикач Out LTCEnable or disable the LTC output stream. When disabled, no audio is sent to the output device regardless of the Thru setting.Вмикає або вимикає вихідний потік LTC. Коли вимкнено, аудіо не надсилається на вихідний пристрій незалежно від налаштування Thru.
Thru dotКрапка ThruClickable toggle in the section header. When ON (and the Out LTC switch is also ON), the raw LTC audio captured from the input device is passed directly to the selected output — no decode / re-encode cycle. This preserves the original signal and eliminates encoder latency. When OFF, the normal timecode encoder output is used. The LTC input source must be running for Thru to produce audio.Клікабельний перемикач у заголовку секції. Це зберігає оригінальний сигнал та усуває затримку кодера. Коли ВИМКНЕНО, використовується нормальний вихід кодера тайм-коду. Джерело LTC-входу повинно бути запущено для роботи Thru.

Custom TriggersВласні тригери

A special group (Layer 0) that appears above the regular clip layers. Custom Triggers are not tied to any specific Resolume clip slot — each one fires a column or a layer/clip combo at a configured time. Useful for triggering global Resolume events that are not individual clips.

Спеціальна група (Шар 0), що відображається над звичайними шарами кліпів. Власні тригери не прив'язані до жодного конкретного слоту кліпу Resolume — кожен запускає стовпець або комбінацію шар/кліп у налаштований час. Корисно для запуску глобальних подій Resolume, що не є окремими кліпами.

Create the first Custom Trigger using the Create Custom Trigger button (appears in the toolbar when no custom group exists). Additional triggers are added with the + button inside the group.

Створіть перший Власний тригер за допомогою кнопки Create Custom Trigger (з'являється на панелі інструментів, коли немає власної групи). Додаткові тригери додаються кнопкою + всередині групи.

File Format (.etr)Формат файлу (.etr)

Easy Trigger uses JSON files with the .etr extension. The file contains a mode field ("settings", "clips", or "all") that tells the loader which sections to restore. Files created with Save settings can be loaded with Load all — the missing clip section is simply skipped.

Easy Trigger використовує файли JSON з розширенням .etr. Файл містить поле mode ("settings", "clips" або "all"), що вказує завантажувачу, які секції відновлювати. Файли, створені за допомогою Save settings, можна завантажити за допомогою Load all — відсутня секція кліпів просто пропускається.

Files are saved to the user's Documents folder by default but can be placed anywhere.

Файли за замовчуванням зберігаються в папці Документи користувача, але можуть бути розміщені будь-де.

OSC Addresses UsedOSC-адреси, що використовуються

Easy Trigger sends the following OSC messages to Resolume when triggering clips or columns:

Easy Trigger надсилає наступні OSC-повідомлення до Resolume при тригерингу кліпів або стовпців:

ActionДіяOSC Address
Trigger clipЗапуск кліпу/composition/layers/[L]/clips/[C]/connect
Trigger columnЗапуск стовпця/composition/columns/[col]/connect

Де [L] — індекс шару (від 1), [C] — індекс кліпу (від 1), а [col] — індекс стовпця (від 1).
