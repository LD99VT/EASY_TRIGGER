Easy Trigger - Full Guide

🌐

UI version: v1.0

1. What this plugin does

  • Receives timecode from LTC / MTC / Art-Net / OSC.
  • Compares incoming time with each clip Trigger Time.
  • When matched, sends an OSC command to Resolume to trigger the target clip.
  • Row highlight: green = active/selected in Resolume, orange = fired by timecode trigger.

2. Left side (Input + Resolume)

  • Timecode Input Settings: choose source type and source parameters.
  • Resolume Settings: IP/Port for outgoing OSC and Input Port for incoming OSC.
  • Sync: connection state indicator (color only).
  • Global Offset: global trigger shift, supports -HH:MM:SS:FF.

3. Right side (Layers/Clips)

  • Each layer can be collapsed with the arrow button.
  • Layer checkbox includes/excludes the whole layer from timecode trigger.
  • Clip checkbox includes/excludes a single clip from timecode trigger.
  • Clip name, Countdown, Trigger Time, Duration.
  • EndClip Event sets post-duration action: Off, Col, L/C.
  • ▶ button (Test) triggers the clip manually.

4. Main buttons

  • Get Clips - requests clips from Resolume and fills the list.
  • Settings - opens save/load menu for .etr files.

5. .etr file format

  • Save Settings: stores only Input/Resolume/UI settings.
  • Save All: stores settings plus all clip trigger rows.
  • Loading a file without clip rows is supported (settings-only restore).
  • Auto-load of the selected file on startup is available.

6. Main OSC addresses

  • Clip offset: /composition/layers/[layer]/clips/[clip]/transport/position/behaviour/offset
  • Clip duration: /composition/layers/[layer]/clips/[clip]/transport/position/behaviour/duration
  • Trigger clip: /composition/layers/[layer]/clips/[clip]/connect

7. Troubleshooting

  • "OSC listener is not running": input port is busy or listener failed to start after loading settings.
  • No clips after Get Clips: verify IP/Port, Input Port, and that Resolume is running.
  • Clip does not trigger: verify layer/clip checkboxes are enabled and Trigger Time format is valid.

Версія UI: v1.0

1. Що робить плагін

  • Приймає таймкод з LTC / MTC / Art-Net / OSC.
  • Порівнює вхідний час із Trigger Time кожного кліпу.
  • При збігу надсилає OSC-команду в Resolume для запуску цільового кліпу.
  • Підсвітка рядка: зелений = активний/вибраний у Resolume, помаранчевий = спрацював по таймкоду.

2. Ліва частина (Input + Resolume)

  • Timecode Input Settings: вибір типу джерела та параметрів входу.
  • Resolume Settings: IP/Port для вихідного OSC та Input Port для вхідного OSC.
  • Sync: індикатор стану зʼєднання (лише колір).
  • Global Offset: глобальне зміщення тригерів, формат -HH:MM:SS:FF.

3. Права частина (Layers/Clips)

  • Кожен шар можна згорнути кнопкою-стрілкою.
  • Чекбокс шару вмикає/вимикає весь шар для тригерів по таймкоду.
  • Чекбокс кліпу вмикає/вимикає окремий кліп для тригерів.
  • Колонки: Clip name, Countdown, Trigger Time, Duration.
  • EndClip Event задає дію після тривалості: Off, Col, L/C.
  • Кнопка ▶ (Test) запускає кліп вручну.

4. Основні кнопки

  • Get Clips - запитує кліпи з Resolume і заповнює список.
  • Settings - відкриває меню збереження/завантаження файлів .etr.

5. Формат файлу .etr

  • Save Settings: зберігає тільки налаштування Input/Resolume/UI.
  • Save All: зберігає налаштування та всі рядки тригерів кліпів.
  • Підтримується завантаження файлу без рядків кліпів (тільки налаштування).
  • Доступне автозавантаження вибраного файлу при старті.

6. Основні OSC-адреси

  • Offset кліпу: /composition/layers/[layer]/clips/[clip]/transport/position/behaviour/offset
  • Duration кліпу: /composition/layers/[layer]/clips/[clip]/transport/position/behaviour/duration
  • Запуск кліпу: /composition/layers/[layer]/clips/[clip]/connect

7. Вирішення проблем

  • "OSC listener is not running": вхідний порт зайнятий або listener не запустився після завантаження налаштувань.
  • Після Get Clips немає кліпів: перевір IP/Port, Input Port і що Resolume запущений.
  • Кліп не тригериться: перевір, що чекбокси шарів/кліпів увімкнені, а формат Trigger Time коректний.
